# ArrayExamples (TypeScript)

This small file demonstrates common array operations in TypeScript: typed arrays, tuples, readonly arrays, generics, map/filter/reduce, and more.

How to run:

1) Using `ts-node` (fast):

```bash
# install if you don't have it
npm install -g ts-node typescript
ts-node "Revision/ArrayExamples.ts"
```

2) Compile with `tsc` then run with `node`:

```bash
npm install -g typescript
tsc "Revision/ArrayExamples.ts" --target ES2019 --module commonjs
node "Revision/ArrayExamples.js"
```
